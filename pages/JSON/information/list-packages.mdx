# List packages
This endpoint lists the packages available for the reseller.


## HTTP Endpoint

`GET https://panel.myownfreehost.net/json-api/listpkgs.php`

## Usage

<Tabs items={['cURL', 'Python']}>
    <Tab>
        ```bash
        curl -u username:password "https://panel.myownfreehost.net/json-api/listpkgs.php"
        ```
    </Tab>
    <Tab>
        ```python
        import requests
        from requests.auth import HTTPBasicAuth

        url = "https://panel.myownfreehost.net/json-api/listpkgs.php"

        response = requests.get(url, auth=HTTPBasicAuth('username', 'password'))

        print(response.json())
        ```
    </Tab>
</Tabs>

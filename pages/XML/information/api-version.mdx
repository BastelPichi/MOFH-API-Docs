# Get API version
This endpoint gets the current API version.


## HTTP Endpoint

`GET https://panel.myownfreehost.net/xml-api/version.php`

## Usage

<Tabs items={['cURL', 'Python']}>
    <Tab>
        ```bash
        curl -u username:password "https://panel.myownfreehost.net/xml-api/version.php"
        ```
    </Tab>
    <Tab>
        ```python
        import requests
        from requests.auth import HTTPBasicAuth

        url = "https://panel.myownfreehost.net/xml-api/version.php"

        response = requests.get(url, auth=HTTPBasicAuth('username', 'password'))

        print(response.text)
        ```
    </Tab>
</Tabs>
